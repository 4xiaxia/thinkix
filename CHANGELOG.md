# Changelog

All notable changes to Thinkix will be documented in this file.

## 0.1.0-alpha

Initial public release candidate.

### Highlights

- Infinite canvas with boards, shapes, text, images, freehand drawing, sticky notes, and mind maps
- AI agent pane that can inspect, create, and edit content on the board
- Mermaid and markdown-to-mindmap insertion flows
- Collaboration support with Liveblocks
- Import/export for `.thinkix`, SVG, PNG, and JPG
- Unit, component, and end-to-end test coverage

### Known Limitations

- Collaboration requires Liveblocks credentials
- Agent web search requires `EXA_API_KEY`
- The AI workflow is optimized for supported diagram and board operations, not arbitrary external actions
- Product polish is still evolving across some advanced flows
