# Security Policy

## Supported Versions

Thinkix is under active development. Security fixes are currently targeted at the latest code on `main`.

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.

Use GitHub's private vulnerability reporting for this repository if it is enabled. If it is not available, contact the maintainers directly before sharing details publicly.

When reporting an issue, include:

- a clear description of the problem
- reproduction steps or a proof of concept
- the affected environment or deployment shape
- any suggested mitigations, if known

We will aim to acknowledge reports quickly, reproduce the issue, and coordinate a fix before public disclosure.
